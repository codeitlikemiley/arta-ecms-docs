title: Issue Creation Process

> Issues are either created by the team member or complainant outside the team Organization.
> Any Issues created from Outside Organization like from Email using Gmail Add-Ons are marked as Triage.
> This is subject for further review or if all requirement is complete can be assigned to designated officer.

## Creating Issue From Email

> A Dedicated Gmail Add Ons , must be created using Google Appscript that is Integrated with Gmail API, OpenAI API, Complaint Form Submission Service.
> This Gmail Add-Ons would be used privately by users that are part of Google Workspace (email that has the same domain name)

Below is the process on how we Create an Issue or Acknowledge an Email.

<Note>A Issue URL is emailed back to user on success, which would allows the complainant to follow up on his/her complaints, and Do other stuff like updaload new documents</Note>

          else Form Incomplete
