## Invitation Process for New Users:

Description:
1. An existing user or admin invites a new user via email.
1. The system generates an invite token and records it in the 'Invite Tokens' table.
1. The invite is sent via email with the token link.
1. The invited user clicks the link, registers, and is associated with the inviting user.
