---
title: Data Export and Backup Process
---

This sequence diagram details the following steps:

1. The user requests access to export data and create backups.
1. The Authentication Service verifies the user's permissions by querying the database.
1. If the user has the necessary permissions, they interact with the Data Export Service to request data export.
1. The Data Export Service retrieves the required data from the database and delivers it to the user.
1. The user then requests data backup through the Backup and Recovery Service.
1. The Backup and Recovery Service initiates the backup process, and the database confirms completion.
1. The Backup and Recovery Service notifies the user of the successful backup.
1. If the user does not have the required permissions, they are denied access to data export and backup functionalities.
