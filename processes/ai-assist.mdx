      ```mermaid
      sequenceDiagram
          participant GmailAddons as Gmail Add-ons
          participant WebForms as Web Forms
          participant Dashboard as Dashboard Issues
          participant IssueStatusAssignment as Issue Status
          participant Decision as Decision Node
          participant AIAutomation as AI Automation
          participant ManualProcess as Manual Process
          participant DocumentManagement as Document Mgmt
          participant IssueResolution as Issue Resolution

          %% Input Channels
          GmailAddons->>IssueStatusAssignment: Assign Triage
          WebForms->>IssueStatusAssignment: Assign To-Do
          Dashboard->>IssueStatusAssignment: Assign Backlog

          %% Issue Status Assignment
          IssueStatusAssignment->>Decision: Evaluate
          Decision-->>IssueStatusAssignment: Not Actionable Violation (Close Issue)

          %% Decision Point
          Decision->>AIAutomation: According to Citizens Charter
          Decision->>ManualProcess: Not Found on Citizens Charter

          %% AI Automation
          AIAutomation->>DocumentManagement: Guide Complainants/Auto-Assign to Support
          AIAutomation-->>IssueResolution: Failure to Comply (Close Issue)

          %% Manual Process
          ManualProcess->>DocumentManagement: Guide Complainants/Assign to Support
          ManualProcess-->>IssueResolution: Failure to Comply (Close Issue)

          %% Document Management
          DocumentManagement->>IssueResolution: Draft Finalized/Not Final
          DocumentManagement-->>IssueResolution: Not Actionable Violation (Close Issue)

          %% Issue Resolution
          IssueResolution-->>IssueResolution: File for Docketing (Closed)
          IssueResolution-->>IssueResolution: Do Not File (Closed)
      ```
