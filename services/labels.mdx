### Labels Management Service

#### Summary

The Labels Management Service provides functionalities for creating, retrieving, updating, and deleting labels. These labels can be associated with various entities in the system, such as issues or tasks, to facilitate organization and categorization.

#### Endpoints

1. **CreateLabel**
    - **Purpose**: Create a new label for categorization.
    - **Request Parameters**:
        - `name` (string): The name of the label.
        - `description` (string, optional): A brief description of the label.
    - **Response**:
        - `label_id` (string): The unique identifier of the created label.

2. **GetLabel**
    - **Purpose**: Retrieve details of a specific label.
    - **Request Parameters**:
        - `label_id` (string): The ID of the label to retrieve.
    - **Response**:
        - `name` (string): The name of the label.
        - `description` (string): The description of the label.

3. **UpdateLabel**
    - **Purpose**: Update the details of an existing label.
    - **Request Parameters**:
        - `label_id` (string): The ID of the label to update.
        - `new_name` (string): The new name of the label.
        - `new_description` (string, optional): The new description of the label.
    - **Response**:
        - `success` (Boolean): Indicates if the operation was successful.

4. **DeleteLabel**
    - **Purpose**: Delete a specific label from the system.
    - **Request Parameters**:
        - `label_id` (string): The ID of the label to be deleted.
    - **Response**:
        - `success` (Boolean): Indicates if the operation was successful.
