### Unified Filters Service

#### Summary

The Unified Filters Service deals with the advanced filtering system outlined in the 'UnifiedFilters' table. It enables the creation, retrieval, modification, and deletion of complex filters, enhancing data view customization and user query experiences.

#### Endpoints

1. **GetFiltersList**
    - Purpose: Retrieves a list of all filters.
    - Request: `GetFiltersListRequest`
    - Response: `GetFiltersListResponse`

2. **GetFilter**
    - Purpose: Retrieves details of a specific filter.
    - Request: `GetFilterRequest`
    - Response: `GetFilterResponse`

3. **CreateFilter**
    - Purpose: Creates a new filter.
    - Request: `CreateFilterRequest`
    - Response: `CreateFilterResponse`

4. **UpdateFilter**
    - Purpose: Updates an existing filter.
    - Request: `UpdateFilterRequest`
    - Response: `UpdateFilterResponse`

5. **DeleteFilter**
    - Purpose: Deletes a filter.
    - Request: `DeleteFilterRequest`
    - Response: `DeleteFilterResponse`
